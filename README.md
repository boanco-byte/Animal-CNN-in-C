# CNN From Scratch in C

Dự án xây dựng CNN từ đầu bằng ngôn ngữ C để tìm hiểu và thực hành cách triển khai các layer, hàm kích hoạt, lan truyền xuôi (forward propagation), lan truyền ngược (backpropagation) và quá trình cập nhật trọng số.

Dự án được thực hiện với mục đích học tập và khám phá cách hoạt động bên trong của CNN, không hướng tới việc huấn luyện hay triển khai các mô hình thực tế.
